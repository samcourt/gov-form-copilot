# System Context

Gov Form Copilot assists users while they complete existing online forms.

## Users

Primary user:

- A person completing a government form.

Secondary users:

- Product teams validating form UX.
- Case workers reviewing evidence-backed suggestions.
- Developers extending the platform.

## External Systems

- Government web forms
- Browser runtime
- Local/API server
- Evidence documents
- Future OCR/LLM services

## Key Boundary

Gov Form Copilot does not own or modify the government system.

It assists the user through the browser while preserving user oversight.
